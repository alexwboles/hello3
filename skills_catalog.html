What is a skill in AgentHub?

A skill is a reusable capability module that can be enabled on one or more agents. Skills define what an agent can do, such as browsing, parsing documents, or scheduling actions.

Web Browsing

Allows the agent to navigate web pages and extract structured findings.

Enabled on 2 agents

Document Reader

Reads PDF and DOCX files and produces concise, cited summaries.

Enabled on 3 agents

Calendar Manager

Creates, updates, and resolves calendar events with conflict checks.

Enabled on 1 agent

Incident Summarization

Builds incident timelines from logs and proposes next response steps.

Enabled on 1 agent
